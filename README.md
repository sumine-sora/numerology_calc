# 数秘術自動計算ツール

生年月日とお名前（ローマ字）から、あなたの6つの主要な数秘術ナンバーを自動で計算するWebアプリケーションです。

## 🌟 特徴

- **6つの数秘術ナンバーを自動計算**
  - ライフパスナンバー（人生のテーマ）
  - ディスティニーナンバー（使命と才能）
  - ソウルナンバー（心の願い）
  - パーソナリティナンバー（周りから見た印象）
  - バースデーナンバー（生まれ持った才能）
  - マチュリティーナンバー（人生後半のテーマ）

- **2つの表示モード**
  - **かんたんモード**：一言で分かりやすい解説
  - **詳細モード**：充実した深い解説

- **レスポンシブデザイン**
  - PC、タブレット、スマートフォンに完全対応

- **マスターナンバー対応**
  - 11, 22, 33を正しく認識・保持

- **プライバシー重視**
  - すべてローカル処理、データは送信されません

### 計算方式

**ピタゴラス式数秘術**を採用

#### アルファベット→数値変換テーブル
```
A=1, B=2, C=3, D=4, E=5, F=6, G=7, H=8, I=9
J=1, K=2, L=3, M=4, N=5, O=6, P=7, Q=8, R=9
S=1, T=2, U=3, V=4, W=5, X=6, Y=7, Z=8
```

#### 母音と子音
- **母音**: A, E, I, O, U
- **子音**: 上記以外のアルファベット

#### 数値還元ルール
- 基本：各桁の数字を足して1桁になるまで還元
- 例外：11, 22, 33（マスターナンバー）は還元せず保持

#### 各ナンバーの計算方法

1. **ライフパスナンバー**
   - 年、月、日をそれぞれ還元
   - 還元された3つの数値を合計して再度還元

2. **ディスティニーナンバー**
   - フルネームの全文字を数値化
   - 合計して還元

3. **ソウルナンバー**
   - 名前の母音（A, E, I, O, U）のみを数値化
   - 合計して還元

4. **パーソナリティナンバー**
   - 名前の子音のみを数値化
   - 合計して還元

5. **バースデーナンバー**
   - 誕生日の日にちを還元

6. **マチュリティーナンバー**
   - ライフパスナンバー + ディスティニーナンバー
   - 合計して還元

### 計算例

**入力**: John Smith（1990年5月15日）

#### ライフパスナンバー
```
年: 1990 → 1+9+9+0 = 19 → 1+9 = 10 → 1+0 = 1
月: 5 → 5
日: 15 → 1+5 = 6
合計: 1 + 5 + 6 = 12 → 1+2 = 3
結果: 3
```

#### ディスティニーナンバー
```
J(1)+O(6)+H(8)+N(5)+S(1)+M(4)+I(9)+T(2)+H(8) = 44 → 4+4 = 8
結果: 8
```

#### ソウルナンバー
```
母音: O(6)+I(9) = 15 → 1+5 = 6
結果: 6
```

## ❓ よくある質問（FAQ）

### Q: 名前は日本語で入力できますか？

A: 現在のバージョンではローマ字のみ対応しています。日本語の名前はローマ字（ヘボン式推奨）に変換してご入力ください。

### Q: マスターナンバーとは何ですか？

A: 11、22、33は特別な意味を持つ「マスターナンバー」として、1桁に還元せずにそのまま扱います。

### Q: データは保存されますか？

A: いいえ、入力したデータはすべてブラウザ上でのみ処理され、サーバーに送信されることはありません。ページを閉じるとデータは消去されます。

### Q: 計算結果を保存できますか？

A: 現在のバージョンでは保存機能はありません。必要に応じてスクリーンショットを撮るか、手動でメモしてください。

### Q: スマホでも使えますか？

A: はい、レスポンシブデザインに対応しているため、スマートフォンやタブレットでも快適に使用できます。

### Q: 計算方法は正確ですか？

A: ピタゴラス式数秘術の標準的な計算方法に基づいており、複数のテストケースで精度を検証済みです。

## 🔍 トラブルシューティング

### 計算結果が表示されない

- ブラウザのJavaScriptが有効になっているか確認してください
- ブラウザのコンソールでエラーが出ていないか確認してください

### エラーメッセージが表示される

- 生年月日がすべて選択されているか確認してください
- 名前がアルファベットとスペースのみで入力されているか確認してください
- 有効な日付（例: 2月30日は無効）か確認してください

### レイアウトが崩れる

- ブラウザを最新版にアップデートしてください
- ブラウザのキャッシュをクリアしてページを再読み込みしてください
